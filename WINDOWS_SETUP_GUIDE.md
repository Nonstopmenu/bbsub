# Гайд: Как запустить проект Bb.sub на Windows

Для того чтобы перенести и запустить этот проект на другом ноутбуке под управлением Windows, выполните следующие шаги.

## 1. Подготовка системы (установка Python)
Сначала нужно убедиться, что на новом ноутбуке установлен Python.
1. Перейдите на официальный сайт [python.org/downloads](https://www.python.org/downloads/).
2. Скачайте последнюю версию Python для Windows (рекомендуется 3.10 или новее).
3. При установке **ОБЯЗАТЕЛЬНО** поставьте галочку в самом низу окна установщика: **`Add Python to PATH`** (Добавить Python в PATH). Это критически важно!

## 2. Перенос проекта
Скопируйте всю папку проекта `BB.sub` с текущего компьютера (через флешку, Telegram, Google Drive или GitHub) на новый ноутбук с Windows.

## 3. Открытие терминала
1. Зайдите в папку с проектом `BB.sub` на Windows.
2. Кликните по пустой области в адресной строке проводника (там, где написан путь к папке), сотрите всё, напишите `cmd` и нажмите `Enter`. Откроется черное окно терминала прямо в этой папке.

## 4. Настройка виртуального окружения
В терминале поочередно вводите следующие команды (после каждой нажимайте Enter):

> [!TIP]
> Виртуальное окружение помогает изолировать библиотеки проекта от всей системы.

1. Создаем виртуальное окружение (папка `venv`):
```cmd
python -m venv venv
```

2. Активируем его:
```cmd
venv\Scripts\activate
```
*(Если всё прошло успешно, слева в терминале появится надпись `(venv)`).*

## 5. Установка библиотек
Теперь нужно установить все необходимые зависимости для работы бэкенда (они все прописаны в файле `requirements.txt`).
В том же окне терминала (с активным `(venv)`) напишите:
```cmd
pip install -r requirements.txt
```
Дождитесь окончания загрузки. 

## 6. Запуск сервера
Все готово! Чтобы запустить сайт (сервер), введите команду:
```cmd
uvicorn main:app --reload --port 8080
```
> [!NOTE]
> Не закрывайте это черное окно, пока хотите, чтобы сайт работал. Если вы закроете его, сервер отключится.

## 7. Проверка
Откройте любой браузер (Chrome, Edge, Safari) и введите в адресную строку:
[http://localhost:8080](http://localhost:8080)

Сайт запустится! База данных (`bb_sub.db`) создастся автоматически, и проект будет работать абсолютно так же, как и на вашем текущем устройстве.
